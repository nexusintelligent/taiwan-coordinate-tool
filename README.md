# 臺灣座標轉換與圖資檢核工具

一套離線優先的 PWA 網頁工具，用於從混合文字中擷取座標、轉換 WGS84 與 TWD97，並在連線時使用國土測繪圖資服務雲確認位置。

## 目前功能

- WGS84（EPSG:4326）與 TWD97 / TM2 zone 121（EPSG:3826）雙向轉換
- 自動判斷 `緯度, 經度`、`經度, 緯度` 與 `X, Y`
- 從含有中文說明、括號與編號的文字擷取多組座標
- 臺灣範圍合理性檢查與明確錯誤訊息
- 自訂 XY、經緯度小數位與進位方式
- 內建及自訂輸出格式
- 一鍵複製結果
- 國土測繪圖資服務雲臺灣通用電子地圖、正射影像
- 可安裝 PWA；斷網時保留轉換、解析和格式化功能

## 線上與離線邊界

座標及輸入文字都在瀏覽器本機處理，不會送到伺服器。國土測繪中心 WMTS 底圖只在連線時載入，且不進行大量或離線快取。

## 本機開發

```bash
pnpm install
pnpm dev
```

## 測試與建置

```bash
pnpm test
pnpm build
```

## 資料來源與聲明

- 座標系統：TWD97 / TM2 zone 121（EPSG:3826）
- 底圖：內政部國土測繪中心國土測繪圖資服務雲

地圖僅供初步位置檢核，正式工程成果仍應以測量資料及主管機關公告為準。
